# Resume Folder

**Drop your resume PDF here and name it `resume.pdf`**

The download button on the portfolio links to:
`./resume/resume.pdf`

Steps:
1. Export your resume as a PDF
2. Name the file exactly: `resume.pdf`
3. Place it in this folder (replacing this README)
4. The download button on the site will automatically work

> If you want a different filename, update the `href` in the hero section HTML
> (search for `resume/resume.pdf` in `index.html` and change accordingly).
